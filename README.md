# 🐾 Cool Cat Kit

Личный дневник и кабинет для кота: хозяин отслеживает здоровье, прививки,
питание и ведёт «историю» питомца. Дизайн — неформальный блокнот: бумажный фон,
рукописный шрифт, скотч, скрепки и наклонённые карточки. Палитра построена от
зелёного `#86a96f` к синим оттенкам.

> Статус: **скелет**. Готовы переходы между страницами, авторизация, заметки,
> календарь и лента «история». Разделы «еда» и «мемы» — заглушки.

## Возможности

- **Вход / регистрация** с загрузкой фото профиля (по умолчанию ставится котик).
- **Главная** — переходы в разделы.
- **Породы** — справочник из 15 пород, карточки + модальное окно с деталями.
- **Медицина и прививки** — записи (не календарь); запись можно привязать к дате.
- **Календарь** — напоминания и заметки по дням; показывает привязанные медзаписи.
- **История кота** — лента с текстом и фотографиями.
- **Профиль** — смена имени, клички кота, фото; выход из аккаунта.
- **Еда** и **Мемы** — заглушки на будущее.

## Как запустить

Это статический сайт, бэкенд не нужен. Из корня проекта:

```bash
python3 -m http.server 8000
```

Открыть <http://localhost:8000>. Можно также выложить на GitHub Pages.

## Структура

```
index.html            вход / регистрация
pages/                внутренние страницы (home, breeds, medical,
                      calendar, history, profile, food, memes)
css/style.css         дизайн-система (палитра, блокнотный стиль)
js/data.js            хардкод-массив пород (от аналитика)
js/app.js             авторизация, шапка, маршрутизатор, localStorage
js/pages.js           логика конкретных страниц
assets/cats/          фото пород и для ленты (webp)
```

## Где хранятся данные

Пока всё в `localStorage` браузера (бэкенда нет): пользователи, сессия,
заметки, события календаря и посты «истории» — изолированы по email.
При необходимости это меняется на FastAPI + БД без переделки фронта.
